# GreyWarden（灰袍守卫）玩家说明

本模组把你的行为转化为“声望”并触发执法系统。
你越偏向犯罪，灰袍纠察队与灰袍警察的追捕会越严厉。

## 1. 安装与启用

1. 将 `GreyWarden` 文件夹放入：`Mount & Blade II Bannerlord/Modules/`
2. 打开启动器，在 `Singleplayer -> Mods` 勾选 `GreyWarden`
3. 建议按官方基础模组在前、`GreyWarden` 在后的顺序加载

建议使用 Bannerlord `1.3.14` 版本游玩。

## 2. 你会遇到的核心机制

- 你的行为会改变“声望”
- 系统每 **2 天** 进行一次全局执法检查
- 声望区间决定是纠察队处理，还是正式警察接管

## 3. 声望区间与执法规则

### 3.1 声望 > 0（守法阶段）

- 每次 2 天检查会发放津贴（按当前正声望计算）
- 已有通缉与追捕会被撤销

### 3.2 声望 -1 到 -10（纠察队阶段）

系统会派纠察队追踪你。接触后你可选：

- `缴纳正式罚金`（谈判窗口）
  - 按当前负声望计算罚金
  - 成功后解除当前通缉并撤队
- `谈判放行`（谈判窗口）
  - 目标金额为正式罚金的 1/4
  - 成功后获得 4 天保护期，声望不变
- `拒绝执法`
  - 直接进入战斗

如果你在战斗中失败或投降：

- 会被押送到定居点执行处罚
- 仅收金币，不再没收背包物品
- 声望按“实缴情金额”恢复，不会强制直接回到 0

### 3.3 声望 <= -11（正式警察阶段）

正式警察会接管追捕。接触后你可选：

- `缴纳正式罚金`（谈判窗口）
  - 成功后清除通缉
- `认罪认罚（赎罪任务）`
  - 任务会出现在任务面板
  - 时限 45 天
  - 探子会持续推送目标情报
  - 击败目标后，可向族长或任意灰袍警察交任务
  - 按接案目标规模恢复声望，最高恢复到 0
  - 失败则额外 -5 声望
- `拒绝执法`
  - 进入战斗，败北后押送处罚（同样按实缴恢复声望）

## 4. 你需要知道的判定细节

- 执法检查是“每 2 天一次”，不是实时每小时刷新
- 谈判放行后有保护期，保护期内不会立刻再刷纠察队
- 进入重罪区后，案件会转交正式警察处理

## 5. 玩法建议

- 想快速脱离追捕：优先缴清正式罚金
- 钱不够时：可走赎罪任务线，用战斗表现恢复声望
- 想稳定拿津贴：保持正声望并避免连续犯罪
