# GreyWarden（灰袍守卫）玩家说明

本模组围绕“灰袍守卫执法系统”展开。  
你在卡拉迪亚的行为会被记录为声望（正/负），并触发不同等级的奖惩、追捕与谈判。

## 一、适用版本与依赖

- 模组ID：`GreyWarden`
- 当前模块版本：`v1.3.14`
- 游戏版本：建议 Bannerlord `1.3.14`（与模块声明一致）
- 依赖模块：
  - `Native`
  - `SandBoxCore`
  - `Sandbox`
  - `CustomBattle`
  - `StoryMode`

建议在启动器中按常规顺序启用官方基础模块，再启用 `GreyWarden`。

## 二、安装方法（玩家）

1. 将 `GreyWarden` 文件夹放入：  
   `Mount & Blade II Bannerlord/Modules/`
2. 打开启动器，在 `Singleplayer -> Mods` 勾选 `GreyWarden`
3. 进入游戏开始游玩

建议新档体验（老档也可用，但跨版本更新后建议先过 1-2 小时游戏时间让系统完成自检和队伍清理）。

## 三、你在游戏中会经历什么

## 1) 声望系统（核心）

- 声望 > 0：偏“守法/善行”
- 声望 = 0：中立
- 声望 < 0：偏“犯罪”

你的战斗和行为会改变声望。系统会根据声望区间自动切换惩罚等级。

## 2) -1 到 -10：纠察队阶段（轻中度违法）

- 系统每 **2天** 检查一次
- 若你在该区间，会派出纠察队追踪你
- 纠察队人数与违法程度相关（违法越重，队伍越大）

当纠察队接触你时，你可选：

- `缴纳正式罚金`：  
  - 按当前规则收取罚款（基础：每点负声望对应固定金额）  
  - 你的通缉/当前惩罚流程结束
- `谈判放行`（谈判界面）：  
  - 价格目标低于正式罚款（当前为正式罚款的 1/4）  
  - 谈判成功后：暂时放行，声望不变  
  - 会获得一段保护期（当前 4 天）
- `拒绝执法/开战`：  
  - 进入战斗流程  
  - 失败或投降后会被押送并执行处罚

## 3) <= -11：正式追捕阶段（重度违法）

- 纠察队不再是主力，转为正式警察追捕
- 会进入更严厉的执法流程
- 罚款标准更高（按重罪规则计算）
- 金币不足时会自动没收物品抵扣

## 4) 被俘惩罚

若在执法战斗中失败（或投降），你会被押送到定居点执行惩罚：

- 罚金结算
- 声望/通缉状态按规则重置
- 执法队会回城并结束本轮任务

## 5) 正声望奖励

当你保持正声望时，可获得持续收益（如日常奖励）并减少被追捕风险。  
系统会优先朝“维稳”方向处理你与灰袍守卫的冲突状态。

## 四、悬赏猎人玩法（进阶）

当你声望达到较高门槛后，会触发灰袍招募线：

- 接受后可进入“悬赏猎人”路线
- 你需要穿戴指定指挥官套装来接取悬赏任务
- 通过追捕并击败目标领取赏金

这是一条“与灰袍合作”的玩法分支，适合想走执法协助路线的玩家。

## 五、常见问题（FAQ）

## Q1：我明明负声望，为什么没有立刻再来追捕？

常见原因：

- 纠察队阶段是“每2天检查一次”，不是实时每小时刷新
- 你刚完成“谈判放行”，保护期内不会立刻再派队
- 你在城内等待后，需要给系统一点时间做队伍回收（通常 1-2 小时游戏时间）


## Q2：为什么我会被没收物品？

当罚款金额大于你持有金币时，系统会按规则没收背包物品抵扣差额。

## 六、未来展望（草案，可讨论）

以下是目前建议的迭代方向（按优先级草案）：

1. `可视化执法日历`：在界面上显示“下次执法检查倒计时/保护期剩余天数”
2. `区域化法律`：不同文化或王国对同类犯罪采用不同罚则
3. `审判链路`：被押送后加入“裁决结果分支”（罚金、劳役、赎身、势力关系影响）
4. `MCM配置`：让玩家可调罚金倍率、检查周期、追捕强度、保护期长度
5. `文本与叙事打磨`：完整中文润色、更多执法对话、事件反馈更清晰


## 七、反馈建议

反馈问题时尽量附上：

- 触发前声望值
- 你选了哪个对话选项（缴罚/谈判/拒绝）
- 是否战败或投降
- 是否在城内等待过
- 大概的游戏时间跨度（过了几天）

这样可以更快定位问题并修复。
